# YouTube Downloader

Simple Python YouTube downloader with:

* Download videos
* Download MP3 audio
* Choose video quality
* Automatic save folder

## Installation

### Step 1: Install Python

Download and install Python from the official website.

During installation, enable:

`Add Python to PATH`

---

### Step 2: Install yt-dlp

Open Command Prompt and run:

```bash
pip install yt-dlp
```

---

### Step 3: Install FFmpeg

Download and install FFmpeg.

After installation, add FFmpeg to your system PATH.

FFmpeg is required for:

* MP3 conversion
* Merging video and audio

---

## Run the Program

Open Command Prompt inside the project folder and run:

```bash
python downloader.py
```

---

## How to Use

1. Paste a YouTube link
2. Select:

   * Video
   * MP3
3. If Video is selected, choose quality:

   * 360p
   * 720p
   * 1080p
   * Best
4. Wait for the download to finish

---

## Download Location

Files are automatically saved in:

```text
C:\Users\<your_username>\Downloads\YouTube
```

---

## Note

Use only for content you own or have permission to download.
